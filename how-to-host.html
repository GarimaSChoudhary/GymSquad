🏋️ How to Host GymSquad

Free hosting on GitHub Pages — takes about 5 minutes, works forever.

1

Create a free GitHub account

Go to github.com and sign up. It's completely free — no credit card needed.

2

Create a new repository

Once logged in, click the + button (top right) → New repository.

Name it something like gymsquad. Set it to Public. Tick "Add a README file". Click Create repository.

3

Upload the app file

Inside your new repository, click Add file → Upload files.

Drag and drop the index.html file you downloaded. Then click Commit changes.

⚠️ The file must be named index.html exactly — not workout-tracker.html. Rename it before uploading if needed.
4

Turn on GitHub Pages

In your repository, go to Settings → Pages (left sidebar).

Under Source, select Deploy from a branch. Under Branch, pick main and folder / (root). Click Save.

Wait about 60 seconds, then refresh the page. You'll see a green box with your live URL:

https://YOUR-USERNAME.github.io/gymsquad/

Replace YOUR-USERNAME with your actual GitHub username.

5

Share the link with your squad

Send that URL to everyone. Each person opens it in their phone browser, creates their account once, and their data is saved permanently.

On iPhone — tap Share → Add to Home Screen in Safari so it feels like a real app.

Important: Because this app uses browser storage (localStorage), each person's data is saved on their own phone. The Squad tab shows everyone who has used the app on the same device. To see each other's stats across phones, the next step would be adding a real backend (like Firebase) — happy to help with that if you want it.
6

Updating the app in future

If you get a new version of index.html, just go back to your GitHub repository, click the file, then click the pencil icon to edit — or use Add file → Upload files again to replace it. Changes go live in about a minute.
